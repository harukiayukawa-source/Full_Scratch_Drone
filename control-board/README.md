# control-boardフォルダの中身について
## KiCadフォルダ
KiCadのプロジェクトのファイルが入っています。このフォルダごとダウンロードしてKiCadで開けば、おそらく読み込めるかと思います。
- drone_v0-00.kicad_pro : プロジェクトファイル
- drone_v0-00.kicad_sch : 回路図ファイル
- drone_v0-00.kicad_pcb : PCBファイル
- lib_drone.kicad_sym : シンボルライブラリファイル
- libfp_drone.pretty : フットプリントライブラリフォルダ

## manufacturingフォルダ
上の基板のgerberのzipファイルが入っています。。JLCPCBなどの基板発注サービスで読み込むことですぐ発注できます。

## docsフォルダ
回路図やPCBの写真、3Dビューア画像とSTEPファイルが入っています。中身の見た目をすぐ知りたい方用です。
